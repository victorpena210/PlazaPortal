# Tenant Management System (Spring Boot)

A full-stack web application for managing tenants in a small office plaza.

## Features
- User registration and login (Spring Security)
- Role-based access (Tenant / Admin)
- Maintenance request system (in progress)
- Online rent payments (Stripe integration planned)

## Tech Stack
- Java
- Spring Boot
- Spring Security
- Thymeleaf
- MySQL
- Flyway
- JPA / Hibernate

## What I Learned
- How authentication works using Spring Security
- Implementing CustomUserDetailsService
- Password encryption with BCrypt
- Structuring a real backend application
